04
HTTP Verification Pass
40 concurrent HTTP workers check every discovered platform URL. Definitive 404 responses are pruned immediately. Timeouts and connection errors are treated as alive — only confirmed dead pages are removed. Runs as the second stage of the pipeline before AI screening.

AI Visual Screening
A pool of three pre-started Firefox instances screenshots surviving platform pages concurrently. Each screenshot is sent to Gemini 2.5 Flash with the URL and username context. Login walls, wrong users, empty accounts, and unrelated pages are removed at medium-to-high confidence. Low-confidence results are kept.

Pipeline

From username
to intelligence
in minutes.

Three independent OSINT pipelines run concurrently the moment you submit a username. Noise is filtered in two automated passes — HTTP and AI — before anything reaches the graph. What you see is verified, structured, and ready to export.

01
Parallel Collection
Maigret + Sherlock + DuckDuckGo dorking run simultaneously via ThreadPoolExecutor. Results are merged into a single graph write.
02
HTTP Verification
40 concurrent workers send HEAD/GET requests to every discovered URL. Definitive 404s are removed immediately.
03
AI Visual Classification
Browser pool screenshots up to 25 platform pages. Gemini classifies each one. Login walls, error pages, wrong users removed at medium+ confidence.
04
Graph Commit + Export
Verified results are committed to SQLite and rendered in D3. Download the complete picture as a structured PDF report.
